Smart Fish Tank

เครื่องควบคุมและให้อาหารปลาอัติโนมัติควบคุมผ่านแอปพลิเคชั่น

บทคัดย่อ

เนื่องจากผู้จัดทำมีปัญหาเกี่ยวกับเรื่องการให้อาหารปลาที่ตรงเวลาและดูคุณภาพน้ำเพื่อจะเปลี่ยนถ่ายน้ำ ผู้จัดทำจึงคิดค้นเครื่องให้อาหารปลาอัตโนมัติ เครื่องวัดความขุ่นของน้ำ และวัดอุณหภูมิน้ำ โดยสามารถควบคุมอุปกรณ์ผ่านสมาร์ทโฟนได้ เพื่อแก้ปัญหาที่เกิดขึ้นและยังจะสามารถเปลี่ยนสีไฟตู้ปลาเป็น สี RGBและมีลูกเล่นหลายแบบ โดยจะใช้ Node MCU V3 ESP8266 , Waterproof Temperature Sensor DS18B20 ,Turbidity sensor water, Ultrasonic Module HC-SR04+, Servo motor Arduino SG90 ผู้จัดทำจึงได้ Smart Fish tank เพื่อจะอำนวยความสะดวก ให้แก่ผู้เลี้ยงปลาในการดูแลได้ง่ายขึ้น
